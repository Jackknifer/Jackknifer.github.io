主题介绍

REDEFINE YOUR WRITING.

"Jackknifer 的个人博客" 是一个基于 GitHub Pages 的轻量写作空间。它借鉴 Redefine 的文档气质：简洁、快速、纯粹，把注意力放回文章本身。

这个首页先复刻 Redefine Docs 的当前页面状态：左侧文档导航，中间正文内容，右侧页面目录，浅灰背景，红色选中态，以及紧凑但清楚的阅读节奏。后续可以继续扩展搜索、归档、标签、评论和深色模式。

Cover 克制的首页

用文档式结构承载博客介绍和最新文章。
